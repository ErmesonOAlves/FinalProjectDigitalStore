# Digital Store

Loja digital feita com React 19, Vite e PrimeReact.

## Setup

### Pré-requisitos

- Node.js 18+
- npm

### Instalação

```bash
git clone https://github.com/seu-usuario/digital-store-final-project.git
cd digital-store-final-project
npm install
npm run dev
```

Acesse em `http://localhost:5173`



## Comandos

```bash
npm run dev     # Desenvolvimento
npm run build   # Build para produção
npm run lint    # ESLint
npm run preview # Preview do build
```

## Stack

- React 19
- Vite
- React Router
- PrimeReact
- React Slick
